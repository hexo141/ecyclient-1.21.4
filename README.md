## - AI-Crafted Experience

ECYclient is a Minecraft client modification **generated by AI**. All code was written by artificial intelligence, with humans providing suggestions, prompts, and feedback. This project represents a collaboration between human creativity and AI capabilities.

We kindly ask for your understanding and acceptance as this project demonstrates what can be achieved when humans guide AI to bring ideas to life.

## 🎨 Beautification Features

### 🎬 Cinematic Splash Video

- **Full-screen startup video** with transparent, rounded-corner window
- **Real-time system monitoring** overlay displaying CPU & Memory usage
- **Live loading logs** displayed elegantly on the video player
- Smooth center-screen positioning with always-on-top behavior

### 🖼️ Dynamic Custom Backgrounds

- **Auto-scanning** of built-in background images from assets
- **Parallax effect** — backgrounds smoothly follow your mouse cursor
- **Automatic rotation** — backgrounds switch every 60 seconds
- **Random selection** on each launch for a fresh experience
- Smooth image loading with proper texture management

### ✨ Animated Title Screen

- **Rainbow animated logo** — "ECYclient" text with flowing HSL color cycling
- **Wave animation** — each character gently oscillates for a lively effect
- **Custom button styling**:
  - Semi-transparent dark backgrounds
  - Rounded corner outlines
  - Elegant hover effects with bright white borders
- **Complete UI overhaul** — singleplayer, multiplayer, language, options, and quit buttons all redesigned

### 🎯 Visual Polish

- **Translucent UI elements** with alpha blending
- **Rounded corners** on splash video window (20px radius)
- **Semi-transparent info panels** with rounded backgrounds
- **Smooth animations** throughout the interface
- **Professional color schemes** with hover state transitions

## 🛠️ Technical Highlights

- **Language**: Kotlin
- **Mod Loader**: Fabric 1.21.4
- **Java Version**: 21+
- **Architecture**: Mixin-based injection system
- **Video Engine**: JavaFX Media Player
- **Rendering**: Native Minecraft rendering pipeline with custom overlays

***

## 🚀 Installation

1. Install **Fabric Loader** for Minecraft 1.21.4
2. Install **Fabric API** and **Fabric Language Kotlin**
3. Place the `ecyclient.jar` in your `mods` folder
4. Launch Minecraft with the Fabric profile
